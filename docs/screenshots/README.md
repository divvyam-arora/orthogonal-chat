# Screenshots

Drop the following PNGs in this folder. They're referenced from the project README:

| Filename | What to capture |
|---|---|
| `hero.png` | Full chat UI mid-turn: header meter visible, sidebar with a few past chats, one assistant reply containing tool cards + streamed text. |
| `tool-cards.png` | A turn that triggered `search_apis` → `get_details` → `run_api`, all three cards expanded showing inputs, outputs, latency, cost. |
| `context-rotation.png` | The amber "context limit reached" banner immediately after a rotation, with the sidebar showing both the original chat and the new "Continued: ..." entry. |
| `budget.png` | The header cost meter with the inline budget editor open (the `$X.XX` cap clicked) so reviewers can see it's editable. |

Recommended width: 1600 px or larger (the README will downscale). PNG, transparent or matching the app background.
